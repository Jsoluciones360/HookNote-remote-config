Política de Privacidad

Última actualización: 31 de enero de 2026

1. Información que recopilamos

No recopilamos información personal directamente, excepto la que el usuario proporcione voluntariamente.

2. Uso de la información

La información se utiliza únicamente para mejorar el servicio y la experiencia del usuario.

3. Cookies

Este sitio no utiliza cookies propias. Servicios de terceros podrían usar cookies según sus propias políticas.

4. Servicios de terceros

La aplicación puede utilizar servicios de terceros que tienen sus propias políticas de privacidad.

5. Seguridad

Tomamos medidas razonables para proteger la información, pero ningún sistema es completamente seguro.

6. Cambios a esta política

Podemos actualizar esta política ocasionalmente. Se recomienda revisar esta página periódicamente.
